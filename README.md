# Global HIV/AIDS Interactive Dashboard

An interactive dashboard for visualizing global HIV/AIDS statistics,
progress, and trends using real public health data.

## Features

- **Real-time HIV Statistics**: Global and regional HIV prevalence, incidence, and treatment data
- **Interactive Visualizations**: Multiple chart types (line, bar, doughnut) showing trends and distributions
- **Regional Analysis**: Compare data across different regions (Africa, Asia, Europe, Americas)
- **Country-Level Data**: Detailed statistics for high-burden countries
- **Progress Tracking**: Monitor progress towards 95-95-95 targets
- **Time Series Analysis**: View trends from 2000 to present
- **Responsive Design**: Optimized for desktop and mobile devices

## Technologies Used

- HTML5, CSS3 
- JavaScript 
- Chart.js for data visualization
- Font Awesome for icons
- Google Fonts (Poppins, Montserrat)

## Data Sources

This dashboard uses data from:
- **UNAIDS** (Joint United Nations Programme on HIV/AIDS)
- **WHO Global Health Observatory**
- **UNICEF HIV/AIDS Data**
- **National HIV/AIDS Programs**
